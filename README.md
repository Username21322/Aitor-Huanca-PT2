## Observaciones y problemas

- Configuración incorrecta de la VM (poca RAM, disco insuficiente o red mal configurada).
- Dependencias de Nextcloud faltantes (extensions PHP, versions incompatibles).
- Apache o MariaDB mal configurados (puertos bloqueados, credenciales erróneas).
- Nextcloud no accesible por IP o firewall.
- Problemas al pujar arxius (límit de mida o permisos del directori data).
- Roles i permisos mal aplicados (editor que no puede editar, visualitzador que puede modificar).
- Enlaces compartidos sin seguridad (sin contraseña o caducidad).
- Acceso desde otras máquinas bloqueado por configuración de red.
- Errores al añadir clouds externos (versiones distintas o permisos insuficientes).
